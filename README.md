# CK3 Murim Chronicles LITE

A Crusader Kings III mod bringing the Murim (martial arts cultivation) world to the medieval era.

## Overview

Murim Chronicles LITE adds cultivation mechanics, sect alignment, and martial arts techniques to CK3. Characters can progress through cultivation realms, learn techniques, and align with sects.

## Installation

1. Extract the zip or copy mod files to your CK3 mod directory:
   - Windows: `Documents/Paradox Interactive/Crusader Kings III/mod/`
   - Linux: `~/.local/share/Paradox Interactive/Crusader Kings III/mod/`
2. Enable the mod in the CK3 launcher.

## Compatibility

- Includes `zz_murim_woc_gui_patch.txt` for Way of Champions compatibility.
