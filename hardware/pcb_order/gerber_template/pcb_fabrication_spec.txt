-------------------

PCB Version:          0.1.0

PCB SPECIFICATIONS
------------------
Board Dimensions:      50mm x 35mm
Total Thickness:       1.6mm ±0.1mm

---------------------
Material Type:         FR-4 TG130

Dielectric Constant:   4.5 @ 1MHz
Copper Weight:         1 oz (35µm) all layers
Solder Mask:           Green LPI (Liquid Photoimageable)
Mask Color Options:    Green, Blue, Red, Black, White, Yellow

DRILL SPECIFICATIONS
--------------------
Minimum Drill Size:    0.2mm (8 mil)
Standard Drill Sizes:  0.2mm, 0.3mm, 0.4mm (staggered vias)

Aspect Ratio:          ≤8:1
Via Tenting:           Yes (recommended for density)
Via Plugging:          No
Castellated Holes:     No

TRACE/SPACE SPECIFICATIONS
--------------------------
Minimum Trace Width:       0.15mm (6 mil)
Minimum Trace Spacing:     0.15mm (6 mil)
Minimum Clearance:         0.15mm (6 mil)
Preferred Trace Width:     0.2mm (8 mil)
EMG Signal Traces:         0.3mm max width
Power Trace Width:         0.5mm (VCC), 0.8mm (VBAT)

SPECIAL FEATURES
----------------
Impedance Control:         No (digital signals only)
Controlled Dielectric:     No
Gold Fingers:              No
Countersink:               No
Counterbore:               No
Edge Plating:              No
Beveled Edge:              No
Assembly Tooling Holes:    Four 2mm holes at corners

COMPONENT CLEARANCES
--------------------
Edge to Copper:            0.3mm minimum
Edge to Component:         0.5mm minimum
Copper to Copper:          0.15mm minimum
Pad to Pad:                0.2mm minimum
Via to Trace:              0.15mm minimum

ZONE REQUIREMENTS
-----------------
[EMG ZONE - Left Side]
- No copper pour under input traces
- Guard ring around high-impedance nodes
- Star ground configuration required
- Separate analog ground return

[MCU ZONE - Center]
- Solid ground plane under MCU
- Short crystal traces
- Adequate decoupling capacitors

[NFMI ZONE - Right-Center]
- Stitching vias around RF coil area
- Keep away from MCU crystal
- Shielding recommended

[AUDIO ZONE - Right Side]
- Separate ground return path
- Filter components close to IC
- Minimize switching noise coupling

FABRICATION NOTES
-----------------
1. Board should be fabricated at standard spec first
2. Panelization: Single units or 2-up panel (2x1)
3. Mouse bites if panelized: 3mm with 5 holes
4. Tooling rails: 5mm on all sides if panelized
5. Fiducials: 1mm diameter, placed in 3 corners

SPECIAL INSTRUCTIONS TO FAB
---------------------------
- Avoid solder mask on EMG guard ring traces
- Silkscreen over copper allowed for component labels
- Part polarity indication required (diodes, LEDs)
- Date code: Yes, bottom layer preferred
- UL/RU markings: Not required
